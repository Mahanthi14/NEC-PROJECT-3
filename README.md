# 🏥 AI-Powered Healthcare Prediction & Resource Management System

## 📌 Overview

The AI-Powered Healthcare Prediction & Resource Management System is a comprehensive healthcare platform designed to improve patient care, hospital administration, and resource optimization through Artificial Intelligence and Data Analytics.

The system integrates disease prediction, patient management, appointment scheduling, electronic health records, treatment recommendations, outcome prediction, hospital resource allocation, analytics dashboards, and AI-powered healthcare assistance into a single platform.

Live Link :https://nec-project-3-wtlb.onrender.com
